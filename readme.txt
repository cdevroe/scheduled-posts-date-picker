Tested up to: 4.6
Stable tag: 0.1.0

== Description ==

This simple plugin adds a date picker to the day input when scheduling a post. This makes it easier to know the day (i.e. monday, thursday) that the date falls on.

== Changelog ==

= 0.2.0 =

* Moved date picker to the bottom of element
* Month and Year are now also set by picker

= 0.1.0 =

* Initial build adds default style jQuery date picker to admin.
